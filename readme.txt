# CSOPESY MCO
How to run?
1. Clone the repository.
2. Open Visual Studio, navigate to the repository's folder and open the `CSOPESY-MCO.sln` file.
3. Open the `CSOPESY-MCO.cpp` file and hit either `F5` or `Ctrl + F5` to compile and run the program.

The main/entry class is `CSOPESY-MCO.cpp`, found in the `CSOPESY-MCO` folder.
